The CandyRush project has undergone significant refactoring toward dependency injection (DI) using VContainer. This audit identifies these anti-patterns and provides actionable refactoring guidance.

Critical

Lines 23-24, 138

Problem: Creates global singleton accessible via static property. Bypasses DI container entirely. Makes testing impossible and creates hidden dependencies across the entire codebase.
